# Vocab Curve Studio v7

A mobile/desktop vocabulary trainer for 100-word SAT-style batches.

## v7 QOL fixes

- Fixed the Daily cap panel so the cap chips, progress bars, and buttons no longer overlap or get cut off.
- Enlarged the desktop study word and answer area so the main recall card uses the empty space better.
- Kept the desktop Study page no-scroll while keeping all five rating buttons visible.
- Kept phone Study mode no-scroll with the fixed bottom rating dock.
- Made the new-card picker more compact and prominent in the desktop side panel.
- Kept review mode strict: word → meaning only. No meaning → word, sentence cloze, or similar-word tests.
- Kept stable save key for version upgrades from the same URL: `vocab-curve-studio:stable`.
- Kept JSON export/import for cross-device progress transfer.

## iPhone use

For the web-app experience on iPhone, host this folder on an HTTPS site, open `index.html` in Safari, then use Share → Add to Home Screen. The standalone HTML can also be opened directly in a browser, but cross-version storage is safest when the hosted URL stays the same.
